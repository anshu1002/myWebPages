WEB DEVELOPMENT

Web browser:- A web browser is application software accesing the World Wide web or a local website. when a user resquests a web page from a particula wensite, the web browser rerives the necessary content from a web server then displays the page of the user's device.

Web Site:- Web site is a collection of web pages and related content that is identified by a common domain name and published on atlead one web server.
Example:- Google, Facebook, Amazon, and Wikipedia.

Web Apps:- It is a application Softwarer that runs on a web browser unlike Software programs that runs locally natively on the operating system of the device. Web applications are delivered on the World Wide Web to users with an active network connection.
Example:- Horde groupware is an open source web application.

HTML

It stand for Hyper Text Markup Language.

HTML 5

World Wide Web Consortium.

In HTML, everything written in tag.

  • < tagName >
  • openning tag- < tagName >
  • closing tag- </ tagName >

    • Content Tag
  • < tag name > content name </ tagName >
    • Empty Tag
  • < tag without content </ tagName >

    • Structure of Web Page

    < !DOCTYPE >
    < HTML >
    < HEAD >information about Page(Metadata- Data about data) < /HEAD >
    < BODY > Content Visible On Page < BODY >
    < /HTML >

    HTML files must be saved with extensions .html or .htm

    < !DOCTYPE >
    < HTML >
    < HEAD >
    < Title > </ Title > < /HEAD >
    < BODY > < /BODY >
    < /HTML >

    • HTML LIST

    HTML supports three types of list

    1. Ordered List
    2. UnOrdered List
    3. Defination List

    In Ordered List and UnOrdered List items are represented by < li >   </ li >

    Example:
    < ol >
    < li > Anshu Mala </ li >
    < li > Neha Kumari </ li >
    </ ol >

    Tag's Attribute

    Every Tag has multiple Attribute.
    Attributes are always written in "Openning Tag.
    Every Attribute has its own value.

    syntax:-

    < Tag Name AttributeName="value" AttributeName="value"

    Attribute value must be provided in "Double Quotes".
    multiple Attribute must be separted by space.

    Every Tag has its own specific Attribute.
    There are four common attribute applicable on every tag.

    1. ID
    2. CLASS
    3. STYLE
    4. TITLE
    1. TITlE:- used to set Tool Tip Text.

    Anchor/Hyperlink

    < a > This tag is used to create Anchor/hyperlink.

    syntax:-

    < a href- 'filePath' > content </ a >
